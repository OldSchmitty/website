TripCo

CS314 Software Engineering final project: A site that solves the travleing salesman problem based on airport locations. Code uses a 3-opt algorithm and calculations are done using geodesic lines through Google Earth. Back end was done in Java via Spark with MySQL database. Front end done in JavaScript using React, NodeJS, and Bootstrap.


Steam User Data Analysis

CS455 Big Data final project: Analyzing a large scale data set. Done using steam user database. Project was done in Java using Spark and a node cluster of 20 computers running HDFS. Created a neural network to create account predictions based on user behavior.


IRC Client/Server

CS457 Computer Networks and The Internet final Project: Set of programs mimicking an IRC chat room server and a client. Not all standards are used, so it is not fully IRC compliant yet. Both the client and server are done in C++ using a thread pools to accomplish tasks. The Client front end is done with Qt.
