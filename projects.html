TripCo
This project was done for CS314 Software Engineering. The basic Idea is to make a site that calculates the shortest trip for a traveler based on airport locations. The user can select and add the airports they plan to fly to and then the code uses a 3-opt algorithm to find shortest paths by calculating geodesic lines to and from each airport. The program then gives the user the shortest path to take based on their selected starting airport. The program also assumes they user does not want to visit any airport twice, and does will not layaway anywhere. The back end was done in java and connects to the front end via Spark. The front end was done in JavaScript using React, NodeJS, and Bootstrap.


Steam User Data Analysis
CS455 Big Data asked tasked us with analyzing a large scale data set. What better set to use than the steam user database. Our initial questioning revolved around account net worth, and how it related to the number of friends and account had. The project was done in Java using Spark and a node cluster of 20 computers running HDFS. We created a neural network to create predictions of how much an account was worth, and how many friends the account has. It turns out that a very large majority of the Steam user base has extremely similar spending habits.


IRC Client/Server
This is a set of programs creating a chat room server and a client for people to connect to it. These programs do not quite comply with IRC standards, but are extremely similar. The standards we set by the class, CS457 Computer Networks and The Internet. Currently the client is fully functional for local network and internet connections. Messages are not currently encrypted, and there is no expectation of privacy as it was not required for the class. Both the client and server are done in C++ and are using a thread pool to accomplish tasks. The Client front end is done with Qt. See the documentation for a list of supported commands.
